# KITTI Raw Data

<dataset-info/>

[数据集链接](http://www.cvlibs.net/datasets/kitti/raw_data.php)

包含38个sequences的kitti raw数据（带有tracklet annotations）。不包含其余不带有tracklet annotations的sequences数据。

每个sequence包含的数据如下（已解压缩）:
- synced+rectified data
- calibration
- tracklets
